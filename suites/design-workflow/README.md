# Design Workflow Suite

Skills for source-first design work: extracting an existing system, committing
to a visual direction, exploring variations, building prototypes, and running a
final polish gate.

## Skills

- `skills/design-workflow/design-workflow/`: suite-level router.
- `skills/design-workflow/design-source-map/`: extract design constraints from
  real sources.
- `skills/design-workflow/design-direction/`: commit a greenfield visual system.
- `skills/design-workflow/design-variations/`: compare meaningful options.
- `skills/design-workflow/design-prototype/`: build stateful interactive
  prototypes.
- `skills/design-workflow/design-polish/`: final review, fix, and verification.

The installable skill directories live under `skills/design-workflow/` for
