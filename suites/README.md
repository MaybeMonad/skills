# Skill Suites

These folders document the categories mirrored under `../skills/`. They are
metadata and navigation only; installable directories with `SKILL.md` live under
`../skills/<category>/<skill>/` for skills.sh and the default `skills` CLI flow.

## Folders

- `unknown-reduction/`: map-vs-territory skills for reducing unknowns before, during, and after implementation.
- `codex-orchestration/`: Codex planning, delegation, verification, and reporting.
- `design-workflow/`: source-first design workflow, visual direction, variations, prototypes, and polish gates.
- `web-reconstruction/`: source-first website cloning and web-effect reconstruction.
- `search-growth/`: search-facing product surfaces, canonical URL policy, indexing inventory, and submission evidence.

## Publishing Contract

Do not add installable `SKILL.md` files here. Keep category membership in
`../catalog/skills.json` and the actual skill directories under `../skills/`.
